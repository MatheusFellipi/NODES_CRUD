# CRUD 

O projeto e um CRUD basico 

## Iniciar o Projeto

* npm install ou yarn.
* OBs: fazer na pasta api tambem para instalar todos as dependencia.

## Primeiros Passos

Siga os passos abaixo dentro da pasta após clonar ou baixar o projeto:

```bash
# Instalar as dependências do projeto
$ yarn install

ou

$ npm install
```

```bash
# Rodar o projeto
$ yarn start

ou

$ npm start
```
